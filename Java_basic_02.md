# Array
- Initialize array
```
int[] arr = new int[3];
/*
int: data type in array is integer
[]: it's an array
arr: array name

new: apply memery space for this newly created array
3: length of array
*/
```
Or...
```
int[] arr = new int[]{1,2,3}
int[] arr = {1,2,3}
```
